# RTL Chat AI - Chrome Extension

یک اکستنشن کروم برای اعمال جهت‌گیری راست‌به‌چپ (RTL) روی عناصر چت.

## ویژگی‌ها

- تبدیل خودکار همه عناصر با `dir="auto"` به `dir="rtl"`
- اعمال `direction: rtl` و `text-align: right` روی این عناصر
- پشتیبانی از عناصر داینامیک (تغییرات DOM)
- تنظیمات پاپ‌آپ برای فعال/غیرفعال کردن
- ذخیره‌سازی تنظیمات
- بازگردانی دقیق عناصری که تغییر داده شده‌اند

## نصب

1. این مخزن را کلون کنید یا دانلود کنید
2. Chrome را باز کنید و به `chrome://extensions/` بروید
3. "Developer mode" را فعال کنید
4. روی "Load unpacked" کلیک کنید
5. پوشه این پروژه را انتخاب کنید

## استفاده

1. پس از نصب، آیکون اکستنشن را در نوار ابزار Chrome پیدا کنید
2. روی آیکون کلیک کنید تا پاپ‌آپ تنظیمات باز شود
3. با استفاده از کلید toggle، RTL را فعال یا غیرفعال کنید
4. تغییرات به صورت خودکار روی صفحات اعمال می‌شود

## فایل‌ها

- `manifest.json` - فایل تنظیمات اکستنشن
- `content.js` - اسکریپت اصلی که استایل‌ها را اعمال می‌کند
- `popup.html` - رابط کاربری تنظیمات
- `popup.js` - منطق پاپ‌آپ تنظیمات

## نکات

- برای استفاده از آیکون‌ها، فایل‌های `icon16.png`, `icon48.png`, و `icon128.png` را در پوشه قرار دهید
- در صورت عدم وجود آیکون، Chrome از آیکون پیش‌فرض استفاده می‌کند
